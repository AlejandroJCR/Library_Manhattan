# Library_Manhattan


Segundo proyecto de Organizacion del Computador se implementa el programa de gestion de una libreria la cual puede agregar, buscar, eliminar, prestar y devolver. Utilizando Hashing y utilizando como Clave Primaria la Cota de cada Libro. Se utilizo prever 2 grupos primarios con capacidad de 3 libros cada uno y 6 grupos de overflow. 

Plataforma: Python con vectores en memoria, pero deben poder grabarse los resultados en disco duro al terminar el programa y leer desde disco los últimos datos guardados cuando éste se inicia.
